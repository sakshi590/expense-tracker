\# Expense Tracker App 💰



\## 📌 About Project

This is a full-stack Expense Tracker application that helps users manage income and expenses.



\## 🚀 Features

\- Add income and expenses

\- Track balance

\- View transaction history

\- User authentication



\## 🛠 Tech Stack

\- Frontend: React / HTML / CSS / JavaScript

\- Backend: Node.js / Express

\- Database: MongoDB
